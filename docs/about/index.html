Adrian:

嗨👋我是Adrian！我和Obliviate是在Bumble上認識的。和這個軟體的字面意思不同，見到Obliviate的第一面，是一種久違的輕鬆感。有關更多我們遇見的故事，請參見： 相遇。

我其實很難表達Obliviate於我是一種什麼感覺，因為這種感覺是若干種奇妙的滋味混合在一起的。

首先是改變。她的出現，可以說像秋風掃落葉一樣的在不到一週的時間改變了我的生活狀態。就在遇到她前，我還非常自信自己在短期內絕對不會談戀愛，我完全做好了長期一個人生活、旅行、工作的準備。我已經習慣了在旅途中和陌生人交朋友，那種完全不需要牽掛的朋友。我覺得不再需要另一個人，不用給對方添麻煩，也不必替對方分擔。但是Obliviate的出現，對我來說是一種難以抵抗的力量。在遇到她後，我不斷的跟自己講“要忍住、要忍住”，但怎麼可能忍得住鴨，我被她和她的難以抗拒完全的吸引了。開始開心她的開心、擔心她的擔心、開始牽掛、開始思念。

然後是青春。其實在遇到她之前，我身邊的朋友都是比我大的哥哥姐姐，自己考慮的事情也被他們帶的很年長，讓我感覺慢慢的在憂慮很多事情，好少年輕的感覺。在遇到Obliviate之前的一段時間我嘗試調整，也在旅途中丟下很多包袱。但直到遇到她，這個00後的小孩er，在太陽宮門口的那個夜晚，我真的有被Obliviate身上的青春感染到，這是一種很幸福的體驗。

竟然還有成長。別看Obliviate是個00後的小孩er，她的內心真的聽成熟的。不是圓滑的那種成熟，而是和她相處，你會感受到她的關心和愛。別看她的戀愛樣本集超少的（2段就夠了），但是跟她相處真的好好好好好有感。沒想到我會用成長來形容自己和Obliviate的相處。坦白的講，從之前的每一段感情中我都學到了很多東西，也成長了很多，但是都是在結束後反思才感受到。但不知為何和Obliviate相處的時間，我感受到自己能夠更會愛一個人了。

感受最多的就是同頻。這個留給Obliviate同學補充哈。

很喜歡一句中二的話：“既有鎧甲，又有軟肋。”但是講真，Obliviate對我來說，只是讓我變得更勇敢，勇敢的不顧一切的去努力愛一個人，因為愛而有鎧甲；因為知道有她而有鎧甲；因為想照顧她而有鎧甲。

好了Obliviate，該你講啦。
